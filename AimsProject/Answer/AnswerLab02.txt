Q1: When should accessor methods be used?
- Retrieving the value of an instance variable for use in a computation or decision-making process.
- Displaying the value of an instance variable to a user through a user interface (e.g., a graphical display or a console output).
- Passing the value of an instance variable as an argument to another method or class.
- Comparing the value of an instance variable to some other value or condition.
